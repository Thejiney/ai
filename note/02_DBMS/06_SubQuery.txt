서브 쿼리는 하나의 SQL 문장의 절 안에 포함된 또 하나의 SELECT 문장
서브 쿼리는 비교 연산자의 오른쪽에 기술하고 반드시 괄호로 둘러 쌓아야 함
서브 쿼리는 메인 쿼리가 실행되기 이전에 한번만 실행
(하나 특수하게 메인쿼리를 기반으로 조건을 검색하는 서브쿼리의 경우가 있음 ex.EXISTS)

단일행 서브쿼리
메인 쿼리의 WHERE 절에서는 단일 행 비교 연산자인 =, >, >=, <, <=, <>를 사용

다중행 서브쿼리
IN, NOT IN, ANY, SOME, ALL, EXISTS
IN(다중행다중열), ANY(SOME)(한개의조건이라도 만족) ALL(모든조건을 만족) EXISTS(서브쿼리 결과가 존재할시 참)
