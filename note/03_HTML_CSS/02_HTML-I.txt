태그
HTML문서(www 웹 구성문서)를 구성하고 있는 요소 
이름과 속성, 속성값으로 이루어짐.
<!DOCTYPE html> -> HTML5 문서임을 인식시켜줌
<html lang-"ko"> -> html 태그의 시작을 알림
<head></head> -> title,css,javaScript 등 환경설정에 관한 정보
글자 관련 태그 : H1(제목),p(본문), br(개행), a(앵커), b, I, sup, del 태그
목록 태그 - ul(순서X) ol(순서O) li(목록요소)
표 태그 - table(표) tr(행) th(제목열) td(일반열) thead(제목부분) tbody(본문) tfoot(하단)
