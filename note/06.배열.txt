배열
하나의 이름으로 여러개의 데이터를 사용.
배열의 크기는 최초에 한번 설정되면 변경이 불가
배열 선언①
int[] iArr = {10, 20, 30, 40, 50};          // 선언
배열의 크기(length)는 5 / 인덱스(순서)는 0~4
배열 선언②
int[] iArr = new int[5];
// 배열선언과 배열 생성(메모리 할당)
배열 선언③
int[] iArr;
배열 변수선언(생성된 배열을 다루는데 사용될 참조변수 선언)

매력적인 함수
System.arraycopy(score, 0, s, 0, score.length) : 배열 깊은 복사
Arrays.toString():배열 값 스트링 출력
